# 📊 Project 1: Titanic Survival Prediction

## 🎯 Objective

Predict whether a passenger survived the Titanic disaster based on passenger information such as age, gender, ticket class, fare, and family details.

### ✨ Features

* Data Cleaning & Preprocessing
* Missing Value Handling
* Categorical Encoding
* Feature Selection
* Train-Test Split
* Random Forest Classification
* Survival Prediction
* Accuracy Evaluation
* Classification Report Generation

### 📁 Dataset Attributes

* PassengerId
* Pclass
* Name
* Sex
* Age
* SibSp
* Parch
* Ticket
* Fare
* Cabin
* Embarked
* Survived (Target Variable)

### 🔄 Workflow

1. Load Dataset
2. Data Cleaning
3. Handle Missing Values
4. Encode Categorical Features
5. Feature Selection
6. Train-Test Split
7. Random Forest Training
8. Prediction & Evaluation

### 📈 Performance

| Metric    | Score  |
| --------- | ------ |
| Accuracy  | 82.12% |
| Precision | 0.82   |
| Recall    | 0.81   |
| F1-Score  | 0.81   |

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Komal-Dhamange/CodSoft.git
```

Move into the project directory:

```bash
cd CodSoft
```

Install required dependencies:

```bash
pip install pandas numpy scikit-learn
```

or

```bash
pip install -r requirements.txt
```
